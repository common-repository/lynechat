A free plugin to add messenger to your website with minimum efforts

== Description ==

You can log in to Lynechat at www.lynechat.com, create a link, and use that link to chat with your users. This plugin
adds a free and configurable messenger to your website. This plugin seamlessly loads the lynechat script and css into your webpage and
adds a messenger button (or icon - can be configured) on the bottom right of your web page.

== Frequently Asked Questions ==

= How do I get the link? =

Login to www.lynechat.com and create one. Click "embed link" and use that link in the settings.

= Can it be configured? =

Yes, there are a few options on the settings page.
